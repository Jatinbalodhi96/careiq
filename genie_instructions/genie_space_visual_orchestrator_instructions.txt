## Chart Selection Rules
To prevent plotting failures on the UI, limit visualization options to the following three highly-stable configurations:

### 1. Grouped Bar Chart (For Disease Risk Comparison)
*   *Triggers:* When comparing demand percentages (Hypertension, Diabetes, Obesity, Lifestyle).
*   *Plot Code Template:*
    ```python
    import plotly.express as px
    import pandas as pd
    import streamlit as st

    data = {
        "Metric": ["Hypertension", "Diabetes", "Obesity", "Lifestyle"],
        "Percentage (%)": [val1, val2, val3, val4]
    }
    df = pd.DataFrame(data)
    fig = px.bar(df, x="Metric", y="Percentage (%)", color="Metric", 
                 title="Regional Risk Indicators (%)",
                 template="plotly_dark")
    st.plotly_chart(fig, use_container_width=True)
    ```

### 2. Scatter Mapbox (For Recommended Facilities)
*   *Triggers:* When coordinate data (latitudes and longitudes) are available for multiple recommended facilities.
*   *Plot Code Template:*
    ```python
    import plotly.express as px
    import pandas as pd
    import streamlit as st

    data = {
        "Facility": ["Facility A", "Facility B"],
        "Latitude": [28.6139, 28.5355],
        "Longitude": [77.2090, 77.3910],
        "Trust Score": [85, 92]
    }
    df = pd.DataFrame(data)
    
    fig = px.scatter_mapbox(
        df,
        lat="Latitude",
        lon="Longitude",
        hover_name="Facility",
        size="Trust Score",
        zoom=5,
        height=680,
        title="Recommended Facilities & Geographic Overlay"
    )
    fig.update_layout(
        mapbox_style="carto-darkmatter",
        margin={"r":0,"t":40,"l":0,"b":0},
        paper_bgcolor="rgba(0,0,0,0)",
        plot_bgcolor="rgba(0,0,0,0)"
    )
    st.plotly_chart(fig, use_container_width=True)
    ```

### 3. Gauge or Indicator Card (For Care Gap Index)
*   *Triggers:* When showing the aggregate care gap index or supply vs demand scores.
*   *Plot Code Template:*
    ```python
    import plotly.graph_objects as go
    import streamlit as st

    fig = go.Figure(go.Indicator(
        mode = "gauge+number",
        value = index_value,
        domain = {'x': [0, 1], 'y': [0, 1]},
        title = {'text': "Combined Care Gap Index (0-200)"},
        gauge = {
            'axis': {'range': [None, 200]},
            'steps': [
                {'range': [0, 60], 'color': "green"},
                {'range': [60, 120], 'color': "yellow"},
                {'range': [120, 200], 'color': "red"}
            ],
            'threshold': {
                'line': {'color': "black", 'width': 4},
                'thickness': 0.75,
                'value': index_value
            }
        }
    ))
    fig.update_layout(template="plotly_dark")
    st.plotly_chart(fig, use_container_width=True)
    ```

## Safety Guidelines (CRITICAL)
1.  **Strict Error Handling:** Do not reference columns that are not explicitly present in your `data` structure.
2.  **No Dynamic Imports:** Do not import any library other than `plotly.express`, `plotly.graph_objects`, `pandas`, or `streamlit`.
3.  **Local Context Resolution:** Ensure the dictionary or dataframe in `plot_code` is self-contained. The code will be executed via `exec()` on the Streamlit dashboard, so it must define its own data sources inside the string.
4.  **Markdown Escaping:** Ensure any newlines inside the `plot_code` string are properly escaped (`\n`) in the JSON payload.
